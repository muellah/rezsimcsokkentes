# rezsimcsokkentes.hu

Rendszerbontó Nagykoncert — 2026. április 10., Hősök tere, Budapest.

30+ előadó. Egy színpad. A kultúra ereje a változásért.

## Deploy

Static single-page site. Deploy to Netlify:

```bash
# Install Netlify CLI
npm i -g netlify-cli

# Deploy
netlify deploy --prod --dir .
```

Or just drag & drop the folder to [app.netlify.com](https://app.netlify.com).
